# <a name="create-and-edit-forms-using-the-form-designer"></a>フォーム デザイナーを使用してフォームを作成および編集する 
[!INCLUDE [cc-beta-prerelease-disclaimer](../../includes/cc-beta-prerelease-disclaimer.md)]

新しいフォーム デザイナーを使用してモデル駆動型アプリのフォームを作成およびデザインします。

> [!IMPORTANT]
> 新しいモデル駆動型フォーム デザイナーは、現在メイン フォームの作成および編集のみをサポートしています。 詳細: [フォームの種類](types-forms.md)

## <a name="create-a-form"></a>フォームの作成 
1. [PowerApps](https://web.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) にサインインします。 
2. 左のナビゲーション ウィンドウで、**データ**を展開してから、**エンティティ**を選択します。 
3. 取引先企業エンティティなどのエンティティを選択してから、**フォーム**タブを選択します。 
4. **フォームの追加**を選択してから、**メイン フォーム (プレビュー)** を選択します。     
    新しいフォームのコンテンツは、既存のメイン フォームの定義を使用して入力されます。 複数のメイン フォームが存在する場合は、フォーム順序のリストの一番上にあるフォームを使用して新しいフォームに入力されます。 
5. 変更内容を保存してアプリのユーザーに対して可視にする場合は、**保存**を選択してフォームを保存または**公開**を選択します。  

## <a name="edit-a-form"></a>フォームの編集 
1. [PowerApps](https://web.powerapps.com/?utm_source=padocs&utm_medium=linkinadoc&utm_campaign=referralsfromdoc) にサインインします。 
2. 左のナビゲーション ウィンドウで、**データ**を展開して選択し、**エンティティ**を選択します。 
3. 取引先企業エンティティなどのエンティティを選択してから、**フォーム**タブを選択します。
4. 編集するフォームを選択してから、**フォームの編集 (プレビュー)** を選択します。  
   または、目的のフォームの隣の **...** を選択してから、**フォームの編集 (プレビュー)** を選択します。 
5. 変更内容を保存してアプリのユーザーに対して可視にする場合は、**保存**を選択してフォームを保存または**公開**を選択します。 

## <a name="add-and-remove-fields"></a>フィールドの追加および削除 
フォームに対してフィールドを追加またはフィールドを削除するには、フィールド ウィンドウを使用します。 フィールド ウィンドウを使用すると、検索およびフィルター処理を実行して、フィールドを素早く検索することができます。 また、使用されていないフィールドのみ表示するオプションが含まれています。 

### <a name="add-a-field"></a>フィールドの追加
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form)
2. フォーム プレビューで、他の既存のフィールドまたはセクションを選択します。 
    - 既存のフィールドを選択すると、新しいフィールドは既存のフィールドの下に追加されます。 
    - セクションを選択すると、新しいフィールドは使用可能なスペースに追加され、フィールドが列全体で等しく配布されます。 
3. **フィールドの追加**を選択、または左のウィンドウで**フィールド**を選択します。  
   フィールド ウィンドウは、既定ではフォーム デザイナーが開くときに開きます。 
4. **フィールド** ウィンドウでは、追加するフィールドを見つけるために、検索、フィルター、またはスクロールします。 
   フィールドが見つからない場合は、フォーム上に既に存在する可能性があります。 既にフォームに追加されたものを含む、すべてのフィールドが表示するには、**未使用のフィールドのみを表示**をクリアします。 
5. **フィールド** ウィンドウで、フォームに追加するフィールドを選択します。 <br />
   または、目的のフィールドの隣の **...** を選択してから、**選択したセクションに追加**を選択します。 
6. 変更内容を保存してエンド ユーザーに対して可視にする場合は、**保存**を選択してフォームを保存または**公開**を選択します。 

### <a name="remove-a-field"></a>フィールドを削除する
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form)
2. フォーム プレビューで、フォームから削除するフィールドを選択します。 
3. **削除**を選択します。 <br />
4. **保存**を選択します。 

    > [!NOTE]
    >   -  フィールドを誤って削除した場合は、**元に戻す**を選択してアクションを戻します。 
    >   -  必須またはロックされたフィールドは削除することができません。 

## <a name="add-and-remove-tabs-and-sections"></a>タブおよびセクションの追加および削除 
フォーム上でタブまたはセクションを追加または削除するには、レイアウト ウィンドウを使用します。 

### <a name="add-a-tab"></a>タブの追加
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form) 
2. フォーム プレビューで、他の既存のタブまたはフォームを選択します。 
    - 既存のタブを選択すると、新しいタブは既存のタブの右に追加されます。 
    - フォームを選択すると、新しいタブはフォームの右端のタブとして追加されます。 
3. **コントロールの追加**を選択、または左のウィンドウで**レイアウト**を選択します。  
4. **レイアウト** ウィンドウで、**2列タブ**などのタブ コントロールを選択して、フォームに追加します。 <br />
   または、目的のタブ コントロールの隣の **...** を選択してから、**フォームに追加**を選択します。  
5. **保存**を選択します。 


### <a name="remove-a-tab"></a>タブの削除
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form)
2. フォーム プレビューで、削除するタブを選択してから、**削除**を選択します。 
3. **保存**を選択します。 
    > [!NOTE]
    >    - タブを誤って削除したときは、**元に戻す**を選択して削除アクションを戻します。 
    >     - フォームには少なくとも一つのタブが必要です。フォーム上の唯一のタブであるタブは削除することができません。 
    >    - ロック済みセクションを持つタブは削除することができません。 
    >    - 必須またはロック済みフィールドがあるセクションを含むタブは削除することができません。 

### <a name="add-a-section"></a>セクションの追加 
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form)
2. フォーム プレビューで、他の既存のセクションまたはタブを選択します。 
    - 既存のセクションを選択する場合、新しいセクションは既存のセクションの下に追加されます。 
    - タブを選択すると、新しいセクションはタブの最初の列の一番下に追加されます。 
3. **コントロールの追加**を選択、または左のウィンドウで**レイアウト**を選択します。
4. **レイアウト** ウィンドウで、フォームに追加するセクション コントロールを選択します。 <br />
   または、目的のセクション コントロールの隣の **...** を選択してから、**選択したタブに追加**を選択します。      
5. **保存**を選択します。 
 

### <a name="delete-a-section"></a>セクションの削除 
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form) 
2. フォーム プレビューで、削除するセクションを選択してから、**削除**を選択します。  
3. **保存**を選択します。 
    > [!NOTE]
    >    - セクションを誤って削除した場合は、**元に戻す**を選択して削除アクションを戻します。 
    >    - タブには各列に少なくとも一つのセクションを含む必要があります。  
    >    - セクションがタブ列内の唯一のセクションである場合、削除することはできません。 
    >    - ロックされたセクションは削除することができません。 
    >    - 必須またはロック済みフィールドがあるセクションは削除することができません。 
 
## <a name="use-the-tree-view"></a>ツリー ビューの使用 
ツリー ビュー ウィンドウには、フォーム上のコントロールおよびフィールドのビジュアル階層が表示されます。 ツリー ビュー内のアイコンは、フィールドまたはコントロールの種類を素早く識別するために役立ちます。 

また、ツリー ビューを使用してフォーム上に存在するフィールドおよびコントロールを選択することができます。 非表示であり、フォーム プレビューで可視ではない要素を選択するときに、ツリー ビューは便利です。 

ツリー ビュー内のノードを展開または折畳み、ノード内の要素を表示または非表示にすることができます。 ツリー ビューで要素を選択するとフォーム プレビューでハイライト表示になり、プロパティ ウィンドウにはその要素のプロパティが表示されます。 

   ![ツリー ビュー](media/tree-view.png)

### <a name="open-the-tree-view"></a>ツリー ビューを開く 
1. フォーム デザイナーを開いてフォームを作成または編集します。 詳細: [フォームの作成](#create-a-form)および[フォームの編集](#edit-a-form)  
2. 左側のウィンドウで、**ツリー ビュー**を選択します。
